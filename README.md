# Real-Time-Object-Detection

Object Detection in Real time using Deep Learning and OpenCV

### Pre-requisite
- OpenCV 3.3+
- Python 3.6+

### to run the code

python real_time_object_detection.py \
	--prototxt MobileNetSSD_deploy.prototxt.txt \
	--model MobileNetSSD_deploy.caffemodel \
  -t "car"
